This build requires 
-
- jdk 
- Intellij IDEA IDE 
- Docker to run 

Software versions
-
- Targets version 25 of java 
- Version 4.0.x of spring boot
- If used with spring boot 26 results will not be the same and will have errors


What is the product built for ? 
- 
- An app in which a user can capture tasks they plan to complete in the future. 

Functionalities
- 
- A user creates a task 
- Add details in the form of a description. 
- They can add a due date by when they aim to complete it by. 
- A priority of the task to rate the task on a scale of low,medium, or high. 


Technique used "User Stories" 
- 
- Agile Development 
- Title : each user story has a title to identify the task 
- Structure : As a , I want , So that....
- Acceptance criteria after user stories. They evalute always to false before the code for creating a task is written. 


What it will do 
- 
- Create a task 
- Update a task 
- Delete a task 
- Complete a task
