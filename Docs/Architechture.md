Two main components 
- 
- Spring boot app 
- User Interface or UI


Interaction 
- 
- The user interacts with the UI in their browser. 
- The UI interacts with spring boot app by making calls to its REST API


Spring boot 
- 
- Data is stored in-memory H2 database 
- data is only saved when application is running


REST API 
- 
- Built on top of HTTP 
- Uses request and response pattern. 
- Controllers will include : Get Put Post Delete
- Uses JSON format in request and responses exchanged
