# Task-Tracker-
Task Tracker application using CRUD operations. Developed using Java and Spring Boot. You can run it on your operating system from the terminal with Maven or any IDE that supports Java. 

This application has helped me stay organized for my daily objectives and goals. 
I built this to enhance my full-stack building abilites. 


You can run with Maven using mvn compile. 


Dependencies used are java spring Security, DataJPA, h2 Databse, starter Webmvc, validation
